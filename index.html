Política de Privacidad — Litro

Última actualización: 27 de marzo de 2026

1. Información que recopilamos

Litro es una aplicación gratuita de consulta de precios de combustible. Recopilamos la mínima información necesaria para ofrecer el servicio:

  • Ubicación del dispositivo: Utilizamos tu ubicación (GPS) exclusivamente para mostrar las gasolineras más cercanas y calcular distancias. La ubicación se procesa localmente en tu dispositivo y no se envía a ningún servidor nuestro.
  • Datos de uso local: Almacenamos en caché local (en tu dispositivo) los datos de estaciones de servicio para mejorar el rendimiento. Estos datos provienen de la API pública del Ministerio de Industria de España (MINETUR).

2. Información que NO recopilamos

  • No recopilamos datos personales (nombre, email, teléfono).
  • No recopilamos datos de navegación o historial de búsqueda.
  • No utilizamos cookies ni tecnologías de seguimiento.
  • No compartimos ningún dato con terceros.
  • No utilizamos servicios de analítica ni publicidad.

3. Permisos de la aplicación

  • Ubicación (ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION): Para mostrar gasolineras cercanas y ordenar por distancia. Puedes usar la app sin conceder este permiso, pero no podrás ver estaciones cercanas.
  • Internet (INTERNET, ACCESS_NETWORK_STATE): Para descargar precios actualizados de la API pública de MINETUR y cargar los mapas de OpenStreetMap.

4. Fuentes de datos

Los precios de combustible se obtienen de la API pública del Ministerio para la Transición Ecológica y el Reto Demográfico del Gobierno de España. Los mapas se proporcionan mediante OpenStreetMap. Litro no es responsable de la exactitud de estos datos.

5. Almacenamiento de datos

Todos los datos se almacenan localmente en tu dispositivo mediante una base de datos interna. No mantenemos servidores que almacenen datos de usuarios. Al desinstalar la aplicación, todos los datos locales se eliminan automáticamente.

6. Seguridad

Las comunicaciones con la API de MINETUR y los servidores de mapas se realizan mediante HTTPS. No almacenamos datos sensibles.

7. Menores de edad

La aplicación no está dirigida a menores de 13 años ni recopila intencionalmente información de menores.

8. Cambios en esta política

Podemos actualizar esta política ocasionalmente. La fecha de última actualización se indica al inicio del documento. Te recomendamos revisarla periódicamente.

9. Contacto

Si tienes preguntas sobre esta política de privacidad, puedes contactarnos en: litroapp@proton.me
